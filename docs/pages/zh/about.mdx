# 关于OpenCGM

<Callout type="warning" emoji="⚠️">
OpenCGM项目及本手册仅提供相关学术信息，不涉及具体的诊疗，不构成任何医疗建议与背书，具体诊疗请寻求医生
</Callout>

OpenCGM由CGM领域工作者[@gaojinhong](https://github.com/gaojinhong)博士在2024年建立。项目致力于共享CGM的相关资料和研究，以帮助大众和研究者进一步深入CGM领域。
