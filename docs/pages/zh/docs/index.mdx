糖尿病是人类健康的重大威胁，持续了解血糖是糖尿病患者日常生活的重要一环。虽然人类对血糖了解的深入，非糖尿病患者也更多的开始关注自身的血糖水平。

对血糖进行管理的先决条件是测量血糖，传统的实验室方法实施复杂。随着自我监测方法的不断发展，在家中及时获取自身的血糖信息成为可能。但是，血糖仪测试仍存在诸如疼痛、复杂等缺点。数十年前诞生的连续血糖监测（CGM）是一项革命性的技术，使个人，尤其是糖尿病患者，能够持续监测他们的血糖水平。

OpenCGM项目希望借此本手册向大众和专业人士提供血糖测量，特别是CGM相关的信息，包括诸如硬件、软件等信息。

在本手册中，您将了解：
- 血糖测量的基本知识
- CGM技术的基本知识
- 用于CGM的硬件、软件、原理、产品路线
- 处理和分析从CGM设备收集的数据，校准CGM设备的各种算法
- 使用本项目中提供的代码的实际步骤
