## 自回归模型

[Real-time glucose estimation algorithm for continuous glucose monitoring using autoregressive models](https://doi.org/10.1177/193229681000400221)

该方法假设某时刻的血糖浓度与前一段时间的血糖有相关关系，遂应用自回归模型从CGM强度信号计算血糖。在验证数据上，整体平均和中位数相对绝对差异（RADs）分别为9.6%和6.7%。符合国际标准化组织（ISO）标准的测量值为88.7%。在低血糖范围内，平均和中位数RADs分别为8.1%和6.0%，符合ISO标准的测量值为86.7%。低血糖检测的敏感性和特异性分别为91.5%和95.0%。

## 自回归

自回归模型（Autoregressive model，简称AR模型）是一种处理时间序列数据的常见统计方法。它的核心思想是利用同一变量过去的值（即历史数据）来预测当前值或未来值，并假设这些历史值与当前值之间存在线性关系。自回归模型特别适用于时间序列数据的预测，能够捕捉数据中的时间依赖性。

自回归模型具有以下几个特点：
1. 简单：模型基于线性关系进行预测，结构简单，易于理解和实现
2. 灵活：可以与其他模型结合，形成更复杂的模型体系，如自回归积分滑动平均模型（ARIMA）等，以提高预测的准确性和稳定性
3. 时间序列预测：特别适用于时间序列数据的预测，能够捕捉数据中的时间依赖性

但是，这种方法也受到一定的限制，比如必须具有自相关，自相关系数是关键。如果自相关系数小于0.5，则不宜采用，否则预测结果极不准确。

## 局限性

在实践中，受自回归模型的限制，该方法的稳定性不足，预测的血糖浓度漂变严重，需要定期用血糖数据校准。
